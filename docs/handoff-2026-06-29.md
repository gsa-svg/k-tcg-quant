## ⚠️ 미해결 핵심 이슈: NM 변형 오매칭 (최우선)

**증상**: 카드 NM(생카드가)에 *틀린 변형* 가격이 붙어있는 경우가 흩어져 있음. 예: "Sanji OP06-119 Manga"가 ¥2,780(일반판가)로 표시 → 실제 망가(スーパーパラレル)는 ¥198,000. 사용자가 직접 발견함. 일본 사이트(유유테이)는 정확하고, **우리 매칭이 틀린 것**.

**오늘 수정**: PRB-02 OP06-119 Sanji Manga ¥2,780→¥198,000 / OP-06 OP06-118 Zoro Manga ¥7,980→¥398,000 / PRB-01 OP06-118 Zoro Manga는 소스 없어 NM 숨김. (PRB 리프린트 Luffy ¥19,800·Kid ¥500, DON Gold는 원래 저가라 정상 — 확인됨)

**아직 전수 검수 안 됨** → 비슷한 오매칭이 더 있을 수 있음. **다음 작업 1순위 = 전 카드(~190) NM 전수 재검증.**

### 전수 재검증 방법 (검증된 매칭 규칙)
1. 유유테이에서 변형가 재수집: yuyu-tei 페이지(`/sell/opc/s/<set>`)에서 **same-origin `fetch`** 로 전 세트(op01~16, st01, eb01~03, prb01~02) 받아 `{번호 → [{name(변형), price}]}` 맵 구성. (브라우저 한 탭에서 fetch 루프; eBay/yuyu 모두 same-origin fetch 가능)
2. 카드 이름의 변형 → 유유테이 변형 매칭:
   - `Red`/`레드` → レッドスーパーパラレル
   - `Manga`/`Super` → スーパーパラレル
   - `Alternate Art`(단독)/`SP` → パラレル
   - `Signature`/`Gold Stamped` → サイン入り
   - `(PRB)` 리프린트 → 유유테이 `(PRB)` 가격(저가, 별도)
   - `DON`/`DON Card` → DON가(저가)
   - 변형어 없음 → base 또는 パラレル
3. 매칭되는 유유테이 가격으로 NM 정정. **유유테이에 해당 변형이 없으면 NM 숨김(추측 금지).**
4. 검증: PSA10(등급) ≥ NM(생카드) 가 일반적. 단 초희귀(레드/슈퍼)는 NM(호가) > PSA10(실낙찰)이 정상일 수 있음.

### 절대 주의
- 일괄(blanket) 수정은 부수 오류를 냄(번호만으로 매칭하면 OP-06 원본과 PRB 리프린트를 동시에 잘못 바꿈). **번호 + 변형 + 세트(원본/PRB)** 까지 구분할 것.
- 불확실하면 숨김. 틀린 값 노출 > 빈 값. (사용자 최우선 가치)
