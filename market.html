FAQ

What is the OPBOX Index?

It is a free equal-weight index of 18 Japanese One Piece Card Game booster boxes, based to 100 on January 7, 2026. Each day it averages every constituent box's price relative to its January 7 value. As of 2026-07-20 it is 157.4 — up 57.4% since January. Sets first tracked after that date (OP-02, OP-15, OP-16) are shown individually but excluded from the index.

What is the Opening Meter?

It counts how many One Piece cards were newly graded by PSA each week — a proxy for how fast sealed product is being opened. The latest verified week is 2026-07-15, when 10,724 cards were graded. Recent weeks may revise as PSA processing completes.

Why is the change measured 'since January', not since launch?

Our daily price tracking began in January 2026. Only a few sets (currently OP-16) were tracked from their actual release, so for every other set we honestly label changes 'since January' rather than claiming a launch-to-now figure we did not measure. For a true since-launch comparison we use the 'vs MSRP' column, which divides the current price by each set's official Japanese launch MSRP.

How many times has each set been reprinted?

Bandai does not publish per-set reprint announcements for the One Piece Card Game, so a definitive count does not exist. What we can show is dated reprint evidence from retailers and distributors (e.g. MediaWorld and other Japanese shops listing 再販 batches with ship dates). The 'Reprints' column counts those dated records; a dash means none were found in our sources, not that a set was never reprinted. Each set page links the sources.
